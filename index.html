Lojas bolsonaro

Este projeto é uma página web simples, desenvolvida inteiramente com HTML e CSS, que apresenta uma coleção de produtos fictícios com o tema “Bolsonaro” a maioria deles sendo tênis e sapatos estilizados. Criado em 2024 como parte da disciplina de Pensamento Computacional, o site tem como objetivo demonstrar os fundamentos de estruturação de conteúdo (HTML) e estilização visual (CSS) de forma prática e criativa.

A página utiliza elementos semânticos do HTML para organizar os produtos em seções claras, com imagens, nomes, descrições e preços fictícios. Já o CSS foi empregado para definir cores, tipografia, espaçamento e layout responsivo, garantindo que o site seja visualmente coerente e funcional em diferentes tamanhos de tela tudo isso sem o uso de frameworks externos, apenas com código puro.
